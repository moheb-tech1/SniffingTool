# Network Packet Analyzer

A Python network traffic analyzer built with PyShark.

## Features
- Capture network packets
- Show source IP and destination IP
- Display highest protocol layer
- Show transport layer information
- Analyze network traffic

## Requirements
- Python 3
- PyShark

## Usage

Install requirements:

- pip install pyshark
- pip install rich
- pip install pyfiglet

## Run
- python3 network-packet-analyzer.py -h

## Disclaimer
This tool is for educational purposes and authorized network analysis only.
